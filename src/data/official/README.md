# Banco Oficial Inicial

Este directorio contiene un banco local inicial para el modo `Ensayo oficial`.

Principios:

- Cada set mantiene trazabilidad al material oficial publicado por DEMRE.
- Los enunciados de este banco inicial corresponden a extractos oficiales de las selecciones publicadas por DEMRE.
- La alternativa correcta se valida contra los clavijeros oficiales publicados por DEMRE.
- La metadata de cada pregunta indica proceso, fecha, fuente y número de pregunta original cuando aplica.

Fuentes base:

- Admisión 2025: https://demre.cl/publicaciones/2025/pruebas-oficiales-paes-regular-p2025.php
- Admisión 2026: https://demre.cl/publicaciones/2026/pruebas-oficiales-paes-regular-p2026
